# 🇩🇪 Berlin Job Market Analysis
### A Data-Driven Guide for International Job Seekers

---

## 📌 Project Overview
This project analyzes 500 real-world data analyst job postings in Berlin to help international job seekers understand the market — built by an international job seeker, for international job seekers.

## 🎯 Key Questions Answered
- What is the average salary for Junior / Mid / Senior analysts?
- Which skills are most in demand?
- Do companies offer Visa Sponsorship?
- Is German language required?
- Which company types hire the most?

## 📊 Key Findings
- **Junior salary:** ~40,000 EUR/year (~3,500 EUR/month)
- **Top skill:** Statistics, followed by Python and SQL
- **Visa Sponsorship:** 24% of companies offer it
- **German required:** 67% of postings

## 🤖 AI Feature
An AI-powered job analyzer that takes any job posting and returns:
- Match percentage for the candidate
- Skills the candidate has ✅
- Skills the candidate lacks ❌
- Recommendation to apply or not
